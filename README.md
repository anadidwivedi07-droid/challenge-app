# Challenge App 🚀

## 📌 Description

This is a Spring Boot backend application for managing monthly challenges.

## ⚙️ Tech Stack

* Java
* Spring Boot
* REST API
* Maven

## 🚀 Features

* Create challenge
* View all challenges
* Update challenge
* Delete challenge

## ▶️ How to Run

1. Clone the repository
2. Open in IntelliJ
3. Run `ChallengeAppApplication.java`
4. Open browser → http://localhost:8080

## 📡 API Endpoints

* GET /challenges
* POST /challenges
* PUT /challenges/{id}
* DELETE /challenges/{id}
